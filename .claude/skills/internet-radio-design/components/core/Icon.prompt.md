The product's complete inline-SVG icon set. Lucide-style 24px stroke icons (globe, search, heart, share, list, chevrons, x, check, copy), the custom radio-wave glyphs (radio, signal), and solid media-transport icons (play, pause, prev, next, volume, volumeMute). Stroke icons inherit `currentColor`.

```jsx
<Icon name="globe" size={20} />
<Icon name="play" size={28} style={{ color: "#fff" }} />
```

- Solid icons (`play`/`pause`/`prev`/`next`/`volume`/`volumeMute`) fill with currentColor; the rest stroke. `Icon.names` lists every name.
